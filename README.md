# Cold Email Generator

A simple tool to generate personalized cold emails using Python, Streamlit, and ChromaDB.

## Features
- Generate professional cold emails for job applications
- Input job description and personal details
- AI-powered content generation

## How to Run
1. Install Python
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app:  
   - Flask: `python app.py`  
   - Streamlit: `streamlit run app.py`
4. Open the link shown in the terminal
